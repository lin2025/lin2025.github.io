10秒轻松打造个性化ChatGPT 3.5版 Permalink

分享\写个网页
2023.03.25 @林同学不姓林

前段时间改写了个 ChatGPT 3.5 接口版网页 下载，作为GPT工具的补充，用着还行，三句话简单的…介绍下：

  1. 零门槛；人人都会，10秒搞定；只有一个 html 文件，在线/本地打开即用，无需部署；轻中度使用不用再去挤官网，无需担心被封；使用 win:记事本 或 mac:文本编辑.app 即可改代码，无需懂技术；界面仿微信；支持上下文记忆、Temperature 设置、token 统计、撤销、重新回答、清空记忆、导出对话记录、对话框自适应高度等功能；电脑端支持 Ctrl + Enter 快捷键发送；长回复一次性接收；小白1分钟轻松完成以下个性化定制：API-Key 、指令、隐形指令、头像、标题、显示比例、字体大小…
  2. AI工具不嫌多，本代码特点是无门槛 + 便捷；省去登录步骤，打开即用，功能上也足以应对日常需求；一秒复制一个GPT，一分钟完成深度个性化定制，能复刻多数GPT小工具的功能；配合 GitHub 、Gitee 等开源代码托管平台，3分钟完成注册&上传，即可随时随地使用云端私有GPT，可上传N个GPT网页（为亲朋好友定制），完全免费。
  3. 有以下缺点：和 ChatGPT 官网一样，国内需使用魔法让手机/电脑出趟国才可以正常使用；接口版本的上下文深度有限制，不适合需要深度记忆的场景；聊天界面不支持 Markdown 、代码高亮、表格线框；目前仅支持 ChatGPT 3.5 gpt-3.5-turbo （理论上升级到 GPT 4.0 应该不难 *仅升级模型，不添加收发图片、插件等功能）。

小站的由来
2023.03.22 @林同学不姓林

首次分享的 GPT 3.5 代码是基于一个 CSDN 博主的 ChatGPT Demo 改写的（原帖已被博主删除/隐藏）。Demo 使用 text-davci-003 模型，界面无样式，只有一个输入框和一个发送按钮。林同学只能看懂点简单代码，之后在使用过程中结合自身的需求，尝试边学边改、增加功能、添加样式、升级为 gpt-3.5-turbo 模型。分享给朋友们试玩，得到的反馈也都还不错，在这个过程中，林同学产生了分享交流的想法…

改写 GPT 代码的过程，林同学产生了分享交流的想法，期间又冒出做个“小站”的想法。随后找到了免费静态网站生成器 Jekyll ，并尝试借用其设计折腾出这个只有一个 Html 页面的静态网站。小站没有后端/后台（过于复杂，看不懂），内容打算以纯手工的方式写入 Html 代码… 好在小站的用途只是记录和导航，内容不多。
